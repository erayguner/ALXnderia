# Security Policy

## Supported Versions

This repository follows the default support policy for the active
production branch. Update this section with specific version support
windows when they are defined.

## Reporting a Vulnerability

Please report security issues privately. Preferred options:

1. Use GitHub Security Advisories for this repository.
2. Email the security contact for your organization.

If you do not have a security contact, create a private advisory and
include a clear description, steps to reproduce, and any known impact.

## Response Expectations

We aim to acknowledge valid reports within 3 business days and provide
status updates until remediation is complete.
